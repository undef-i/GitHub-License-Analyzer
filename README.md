# GitHub License Analyzer

A modern web application for analyzing GitHub repository licenses with interactive visualizations.

## Development Setup

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
